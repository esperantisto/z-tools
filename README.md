# z-tools 
Linux toolbox under development


## Everything is born

## Installation

* mkdir build
* cd build
* cmake .. 
* make
* ./z-tools

## Build deb or tar.gz

* mkdir build
* cd build
* cmake .. ***or*** cmake .. -DBUILD_TYPE=TGZ
* make packae

*You get the file "\*. deb" or "\*.tar.gz"*
